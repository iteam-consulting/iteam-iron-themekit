Iron Theme

Layout

The Iron Theme can be used by adding the "layout" class to the root element of your website (i.e. the body element). All of the optional classes discussed below should be added/removed from the same root element.

Responsiveness

All mobile devices will use a stacked layout, with the navigation being a toggleable menu. A menu icon button should be used to toggle the "show-menu" class. This menu exposes itself by pushing the content to the right. It closes in the opposite direction.

The Default Layout

No class is needed to use this layout. This layout has a stacked header, nav, main, and footer. You have may have the optional "fixed-header" class to fix the header to the top. The navigation will hide while scrolling down, and be shown while scrolling up.

The "Split Header" Layout

Alongside the "layout" class, use the "split-header" class. This layout is the same as the default layout, but the header and nav are inline with each other. You may add the optional "left-nav" class to align the nav items to the left.

The Sidebar Layout

Alongside the "layout" class, use the "sidebar" class. This layout has the header on top, nav and main inline with each other, and the footer at the bottom of the nav.
